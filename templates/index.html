The PBS Mapping Tool

Part I: Run the CCL scripts and save the output

1. Click the button below to copy the code to your clipboard, then paste in DVDev and run the script

Save the PBS Items to a xlsx spreadsheet called 'PBS Items' to load in later. Make all the excel cells 'Text' rather than 'General' in case Excel automatically changes date formats etc

2. Click the button below to copy the code, cntl-v in DVDev and run the script

Save the Pharmacy Catalog to a xlsx spreadsheet called 'Pharmacy Catalog' to load in later.

3. Click Next to begin mapping



NOTES ON USE:

  • Single click buttons in this app (double clicking might cause issues)
  • At the End, after the update code is stored in your clipboard you can simply close the window to shut down the app
  • Care has been taken to have the desired functionality, but can't be guaranteed, so look over the final update code
